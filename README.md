# NIVEA — Brief y material para web de agencia de tours náuticos

Material de referencia para construir el sitio web de una **agencia de tours náuticos**
en la zona de **Santa Catalina / Coiba (Panamá)**. Este repo reúne el brief del cliente
(audio transcrito), las fotos y los videos que servirán de contenido visual.

> **Para el modelo de IA que use este repo:** la fuente de verdad del *qué quiere el cliente*
> es [`transcripcion-audio.txt`](transcripcion-audio.txt). Las carpetas `fotos/` y `videos/`
> son el material visual real (no usar placeholders). Los nombres de archivo describen su contenido.

## Qué quiere el cliente (resumen del audio)

Una agencia que **no solo venda tours a Coiba**, sino una oferta más amplia:

- Tours a **Coiba** y **otras islas**.
- Tours de **playa**.
- Tour de **pesca**.
- **Avistamiento de ballenas** (*whale watching*).
- **Visitar la costa cerca de Santa Catalina** — pueblos costeros que *nadie más ofrece*
  como tour, pero que a los turistas les encanta.
- **Paseo en bote al atardecer** (*sunset boat ride*) — oferta de verano, nadie más lo hace.

### Requisitos de la web mencionados

- **Encabezado (hero)** con el **nombre de la agencia** y debajo el subtítulo
  **"Boat Adventures"** (Aventuras en bote).
- Estructura libre / a criterio del diseñador — el cliente espera una propuesta que lo sorprenda.

### Notas importantes

- **El nombre "NIVEA" es provisional.** El cliente lo va a cambiar; aún están evaluando opciones.
  No anclar el branding a "NIVEA".
- El cliente hace flyers/posters para Instagram; las **fotos de este repo son parte de ese material**.
